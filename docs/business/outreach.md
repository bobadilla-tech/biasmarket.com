# Bias Market — 50 Leads in 7 Days (Outreach Playbook)

## 🎯 Objective

Get **50 qualified GOM leads** (not just contacts) who:

- Run group orders
- Handle 30+ orders per drop
- Use manual payments

---

## 🧠 Definition of a “Lead”

Count someone as a lead ONLY if:

- They actively run group orders
- You’ve contacted them
- They responded OR clearly fit ICP (high confidence)

---

## 📍 Channels (Prioritized)

1. Instagram → **Primary (60%)**
2. Twitter/X → **Secondary (30%)**
3. Telegram/Discord → **Opportunistic (10%)**

---

## 🔍 Day 1–2: Build Your Lead List (50–80 targets)

### Instagram Search Queries

Search these exactly:

```txt id="y9glin"
"group order"
"GO manager"
"photocard GO"
"kpop go"
"album go"
```

---

### What to Look For (filter HARD)

Only include accounts that:

- Have **order forms in bio**
- Show **recent drops (last 30 days)**
- Have **highlight stories (GO, orders, proofs)**
- Show **payment instructions posts**

---

### Disqualify immediately:

- <10 orders visible
- inactive accounts
- no proof of selling

---

## 📊 Build a Simple Tracker

```txt id="5i2n3z"

```

Goal: → 50–80 high-quality prospects

## ⚡ Day 2–5: Outreach (Volume + Personalization)

### Daily Target:

- 15–25 DMs/day
- Total: ~80–100 messages

---

## 💬 DM Script (High Conversion)

### Version 1 (short, direct)

```txt id="e5c1sa"
Hey — saw you're running GOs 👀

Quick question:

How are you currently tracking payments + order confirmations?

I’m building a tool specifically for GOMs to:
- track orders automatically
- collect payment proofs cleanly
- reduce DM chaos

Would love to let you try it for your next drop.
```

---

### Version 2 (pain-focused)

```txt id="9p7v0x"
Hey — random but I noticed you're running group orders

Quick question:

Do you still use Google Forms + spreadsheets to track everything?

I'm building something to replace that workflow (especially payment proof tracking)

Happy to set it up for you if you’re interested
```

---

## ⚠️ Messaging Rules (THIS matters)

- Keep it **short**
- Ask a **question**
- Don’t pitch too hard
- Don’t sound like a startup

---

## 🔥 Response Handling

### If they say:

**“Yes / still using forms”**

Reply:

```txt id="k1o6x4"
Makes sense — that gets messy fast

I built a simple storefront where:
- users place orders
- upload payment proof
- everything is tracked in one place

If you want, I can set one up for you in a few minutes
```

---

### If they show interest:

→ Move to:

- quick demo (Loom or live)
- or manual setup for them
